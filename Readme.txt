El proyecto se centra en un sistema de seguimiento de stock para una empresa agrícola que utiliza y recibe productos que se aplican en el cultivo.
Permite consultar el stock actual, recibir productos comprados que se suman al stock, y registrar uso de productos.
Asimismo, el usuario (empleado de finca) que está por utilizar un producto en una pulverización, puede consultar el clima para ver si las condiciones son buenas para realizar la misma.
